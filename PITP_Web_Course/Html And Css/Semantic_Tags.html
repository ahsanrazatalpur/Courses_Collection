Semantic Tags

Semantic Tags are those who clearly define their meaning and purpose to the developer and browser

Semantic tag mean meaningfull HTML structured

Without semantic tags we have to make 3 div with class header , main, footer

With Semantic Tags we use built-in semantic tags header, main, footer

It also help to enhamce SEO in our webs

The Common Semantic Tags are

  • nav (used for nav-links)
  • aside (used for sidecontent, sidebar, ads)
  • main (wrap main content)
  • section (wrap multiple related content)
  • article (used to write article)
  • header (Used for define heading , top content)
  • footer (used for footer or bottom content)
  • time (usedd to represent time)
  • mark (mark or highlight important content)
  • figure (img or diagram with caption)
