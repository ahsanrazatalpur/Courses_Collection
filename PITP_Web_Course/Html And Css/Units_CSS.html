Units in CSS

px (pixels): Absolute unit. 1 inch = 96px. Example: width: 100px;

vw (viewport width): 1vw = 1% of viewport width. Example: width: 50vw;

vh (viewport height): 1vh = 1% of viewport height. Example: height: 100vh;

rem (root em): Relative to root element font-size. Default is 16px. Example: font-size: 1.5rem;

cm (centimeter): Example: width: 5cm;

mm (millimeter): Example: width: 5mm;

in (inch): Example: width: 2in; (1in = 96px)
