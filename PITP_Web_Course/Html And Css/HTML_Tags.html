HTML Tags, Attributes, and CSS Properties Cheat Sheet

  • <!DOCTYPE html> declares the document type
  • <html> root element
  • <head> contains metadata, <title>, <style>, <link>
  • <body> main content of the page
  • <h1> - <h6> heading tags
  • <p> paragraph
  • <br> line break
  • <hr> horizontal rule
  • <ul>, <ol>, <li> lists
  • <a> anchor tag, attribute href, target
  • mailto: for email links
  • <img> element with src, alt, width, height
  • <video> and <audio> media elements
  • <iframe> embedding other pages or content
  • <strong>, <b> bold text
  • <em>, <i> italic text
  • <mark> highlighted text
  • <small> small text
  • <del> deleted text
  • <ins> inserted text
  • <sub>, <sup> subscript/superscript
  • <header>, <footer>, <nav>
  • <section>, <article>, <aside>
  • <main>, <figure>, <figcaption>
  • <table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot>
  • <form> element with attributes action, method
  • <input> types: text, password, email, number, checkbox, radio, submit
  • <textarea>, <select>, <option>, <button>
  • id, class, title, alt, value, placeholder
  • color, background-color, font-size, font-family, margin, padding, border
  • display, flex, justify-content, align-items, position, top, left, right, bottom
  • overflow, z-index, text-align, line-height, box-shadow, border-radius
