HTML Elements Reference

Block Elements

Name Meaning Use
<div>DivisionGeneric container for grouping content
<p>ParagraphDefines a paragraph of text
<h1> - <h6>HeadingsDefines headings from largest to smallest
<ul>Unordered listCreates a bulleted list
<ol>Ordered listCreates a numbered list
<li>List itemDefines an item inside a list
<dl>Description ListContainer for terms and descriptions
<dt>Description TermDefines a term in a description list
<dd>Description DetailsDefines the description of a term
<table>TableDefines a table
<thead>Table HeadGroups header content in a table
<tbody>Table BodyGroups body content in a table
<tfoot>Table FooterGroups footer content in a table
<tr>Table rowDefines a row in a table
<td>Table dataDefines a cell in a table row
<th>Table headerDefines a header cell in a table row
<form>FormContains form elements for user input
<textarea>Text areaMulti-line text input in forms
<header>HeaderDefines header section of a page or section
<footer>FooterDefines footer section of a page or section
<section>SectionDefines a thematic section of content
<article>ArticleDefines a self-contained article
<aside>AsideDefines content aside from main content
<nav>NavigationContains navigation links
<blockquote>Block quoteDefines a long quotation
<pre>PreformattedPreserves whitespace and line breaks
<figure>FigureContainer for media and caption
<details>DetailsExpandable/collapsible content
<noscript>No ScriptDisplays content if JavaScript is disabled
<map>Image MapDefines clickable areas on an image using <area>
<area>AreaDefines a clickable region inside <map>

Inline Elements

Name Meaning Use
<span>SpanGeneric inline container
<a>AnchorDefines a hyperlink
<strong>Strong textDefines important text (bold)
<em>EmphasisDefines emphasized text (italic)
<b>BoldDisplays text in bold
<i>ItalicDisplays text in italic
<u>UnderlineUnderlines text
<small>SmallDisplays smaller text
<abbr>AbbreviationDefines an abbreviation with tooltip
<acronym>AcronymDefines an acronym (deprecated, use <abbr>)
<dfn>DefinitionRepresents a term being defined
<code>CodeDisplays inline code snippet
<tt>TeletypeDisplays text in monospace font (deprecated)
<q>Inline quoteDefines a short quotation
<img>ImageEmbeds an image
<input>Input fieldCollects user input in forms
<label>LabelDefines label for a form input
<mark>HighlightHighlights text
<sub>SubscriptDisplays text below baseline
<sup>SuperscriptDisplays text above baseline
<time>TimeRepresents date/time
<var>VariableRepresents variable in text or code
<kbd>Keyboard InputRepresents user input from a keyboard or device
<bdo>Bi-directional overrideOverrides text direction
<output>OutputRepresents result of calculation
