HTML Entities Cheat Sheet

HTML entities are used to display special characters as text in the browser, instead of being interpreted as HTML.

Character HTML Entity Description / Use
< &lt; Represents the “less than” symbol (<). Use when you want to show < in text.
> &gt; Represents the “greater than” symbol (>). Use when you want to show > in text.
& &amp; Represents the ampersand (&). Needed to prevent HTML from misreading & as a code start.
" &quot; Represents a double quote (") inside HTML attributes or text.
' &apos; Represents a single quote (') inside HTML attributes or text.
  &nbsp; Non-breaking space. Used to add extra spaces without collapsing.
© &copy; Copyright symbol.
® &reg; Registered trademark symbol.
™ &trade; Trademark symbol.
¢ &cent; Cent symbol.
£ &pound; Pound currency symbol.
¥ &yen; Yen currency symbol.
€ &euro; Euro currency symbol.
• &bull; Bullet point.
… &hellip; Horizontal ellipsis (…) for truncated text.
§ &sect; Section symbol.
¶ &para; Paragraph symbol.
× &times; Multiplication symbol.
÷ &divide; Division symbol.
≤ &le; Less than or equal to.
≥ &ge; Greater than or equal to.
± &plusmn; Plus-minus symbol.
½ &frac12; Fraction one-half.
¼ &frac14; Fraction one-quarter.
¾ &frac34; Fraction three-quarters.
← &larr; Left arrow.
→ &rarr; Right arrow.
↑ &uarr; Up arrow.
↓ &darr; Down arrow.
