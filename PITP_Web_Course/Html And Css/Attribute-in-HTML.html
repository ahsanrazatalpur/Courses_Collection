Common HTML attribute

  • class (used to give class to element)
  • id (used to give id to element)
  • src (used to provide path or img path)
  • alt (add extra meaning to img)
  • title (used to give title)
  • style (used to give inline styling)
  • placeholder (used to default text in input)
  • disabled (use to diabled any element eg: in forms)
  • readonly (make contanet uneditable)
