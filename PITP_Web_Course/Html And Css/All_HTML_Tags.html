HTML Tags Reference

Name Meaning Use
<!DOCTYPE>Document Type DeclarationDefines the HTML version of the document
<html>Root elementWraps the entire HTML document
<head>Document headContains metadata, title, scripts, and styles
<title>Document titleSets the title of the webpage shown in the browser tab
<meta>MetadataDefines character set, viewport, description, etc.
<link>External resource linkLinks CSS files or icons
<style>CSS stylesInternal CSS styles for the document
<body>Document bodyContains the visible content of the page
<h1> - <h6>HeadingsDefine headings from largest (<h1>) to smallest (<h6>)
<p>ParagraphDefines a block of text
<br>Line breakInserts a single line break
<hr>Horizontal ruleCreates a thematic break (horizontal line)
<a>AnchorCreates a hyperlink to another page or location
<img>ImageEmbeds an image into the webpage
<ul>Unordered listCreates a list with bullets
<ol>Ordered listCreates a numbered list
<li>List itemDefines an item inside <ul> or <ol>
<dl>Description ListContainer for terms and descriptions
<dt>Description TermDefines a term in a description list
<dd>Description DetailsDefines the description/details of a term
<table>TableDefines a table
<thead>Table HeadGroups header content in a table
<tbody>Table BodyGroups body content in a table
<tfoot>Table FooterGroups footer content in a table
<tr>Table rowDefines a row in a table
<td>Table dataDefines a cell in a table row
<th>Table headerDefines a header cell in a table row
<form>FormContains form elements for user input
<input>Input fieldCollects user input in forms
<textarea>Text areaMulti-line text input in forms
<button>ButtonClickable button in forms or scripts
<select>DropdownAllows users to select an option from a list
<option>OptionDefines an option inside <select>
<iframe>Inline frameEmbeds another HTML page inside the current page
<script>ScriptContains JavaScript code or links to external JS
<noscript>No ScriptDisplays content if JavaScript is disabled
<div>DivisionGeneric block container for content or layout
<span>SpanGeneric inline container for text or elements
<strong>Strong textDefines important text (bold)
<em>EmphasisDefines emphasized text (italic)
<small>Small textDefines smaller text size
<code>CodeDefines inline code snippet
<pre>Preformatted textPreserves whitespace and line breaks
<q>Inline quotationWraps a short inline quote
<blockquote>Block quotationIndents a section as a quote
<abbr>AbbreviationDefines an abbreviation with optional full text tooltip
<acronym>AcronymDefines an acronym (deprecated, use <abbr> instead)
<dfn>DefinitionRepresents a term being defined
<var>VariableRepresents a variable in programming or math
<kbd>Keyboard InputRepresents user input from a keyboard or other device
<tt>TeletypeDisplays text in monospace font (deprecated)
<map>Image MapDefines clickable areas on an image using <area>
<area>AreaDefines a clickable region inside <map>
